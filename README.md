> 
> 📚 **适合人群**：有编程基础的开发者（Java/Python/其他语言均可）
> 
> ⏰ **学习周期**：5-6个月，每天2-3小时
> 
> 💼 **就业方向**：AI应用开发工程师、大模型应用工程师、AI解决方案工程师

✅ **技能1：大模型微调技术栈**
- P-Tuningv2、LoRA系列微调
- DeepSpeed框架
- 3D并行训练
- 三段式训练（SFT、Reward Model、PPO/DPO/GRPO/GSPO）
- **完整数据处理流程**

✅ **技能2：Agent智能体开发**
- LangChain框架
- LangGraph框架
- RAG技术
- 知识图谱

✅ **技能3：工程化落地能力**
- Docker封装
- FastAPI服务
- ONNX技术
- vLLM推理加速
- SGLang推理加速
