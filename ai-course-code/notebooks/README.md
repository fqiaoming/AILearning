# Jupyter Notebooks

本目录包含课程的交互式Notebook，适合边学边练。

| 文件 | 说明 | 对应章节 | 状态 |

## 使用方法

```bash
# 1. 安装Jupyter
pip install jupyter

# 2. 启动Jupyter
jupyter notebook

# 3. 打开对应的.ipynb文件
```

## 适合场景

- **学习新概念**：边看讲解边运行代码
- **实验API**：快速测试不同参数的效果
- **调试代码**：逐步执行，查看中间结果
- **课堂演示**：边讲边演示

## 注意事项

1. 运行前确保已安装所有依赖
2. 配置好环境变量（.env文件）
3. API相关notebook需要启动LM Studio或配置OpenAI Key
