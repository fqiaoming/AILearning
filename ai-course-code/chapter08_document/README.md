# 第8章：文档处理工程化

本章代码展示各种文档的加载和处理，为RAG系统准备数据。

## 文件说明

| 文件 | 说明 | 对应课程 |
|------|------|----------|
| 01_document_loaders.py | 文档加载器 | 第46课-文档加载 |
| 02_text_splitters.py | 文本分割器 | 第47课-文本分割 |
| 03_pdf_processing.py | PDF处理 | 第48课-PDF处理 |
| 04_web_scraping.py | 网页抓取 | 第49课-网页处理 |

## 使用方法

```bash
# 安装依赖
pip install langchain-community pypdf unstructured beautifulsoup4

# 运行示例
python 01_document_loaders.py
python 02_text_splitters.py
```
