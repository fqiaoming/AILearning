## 核心概念

1. **全参数微调**：更新所有参数，效果好但资源消耗大
2. **LoRA微调**：只更新少量参数，高效低成本
3. **数据准备**：数据质量决定微调效果
4. **评估方法**：Loss、准确率、人工评估
