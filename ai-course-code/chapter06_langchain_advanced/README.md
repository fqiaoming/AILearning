# 第6章：LangChain高级

本章代码展示LangChain的高级功能：Memory、Router、Callback等。

## 文件说明

| 文件 | 说明 | 对应课程 |
|------|------|----------|
| 01_router_chain.py | 路由链 | 第31课-RouterChain |
| 02_memory.py | 对话记忆 | 第32课-Memory与对话管理深入 |
| 03_callback.py | 回调系统 | 第33课-Callback系统与Chain监控 |
| 04_debugging.py | 调试技巧 | 第34课-Chain调试技巧 |
| 05_optimization.py | 性能优化 | 第35课-Chain性能优化 |
| project_chatbot/ | 实战项目：多功能聊天机器人 | 第36课-综合实战项目 |

## 使用方法

```bash
# 运行示例
python 01_router_chain.py
python 02_memory.py
python 03_callback.py

# 运行实战项目
cd project_chatbot
python chatbot.py
```

## 前置条件

1. 已完成第5章学习
2. 已安装依赖：`pip install langchain langchain-openai`
