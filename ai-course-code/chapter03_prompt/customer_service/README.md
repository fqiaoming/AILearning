# 智能客服实战项目

第3章第15课的完整实战项目 - 从零搭建一个智能客服系统。

## 项目结构

```
customer_service/
├── README.md              # 本文件
├── intent_classifier.py   # 意图分类器
├── response_generator.py  # 回复生成器
├── bot.py                 # 主程序
└── test_intent.py         # 测试脚本
```

## 功能特点

1. **意图识别**：识别6类用户意图
   - 订单查询
   - 退换货
   - 商品咨询
   - 投诉建议
   - 账户问题
   - 其他

2. **智能回复**：根据意图生成个性化回复

3. **转人工判断**：复杂问题自动转人工

## 使用方法

```bash
# 1. 确保已配置环境变量
# 2. 启动LM Studio

# 3. 交互式测试
python bot.py

# 4. 单独测试意图分类
python intent_classifier.py

# 5. 单独测试回复生成
python response_generator.py
```

## 示例对话

```
用户：我的订单什么时候到？
[意图：order_query，置信度：0.90]
客服：您好！您可以在"我的订单"页面查看物流详情。一般情况下，普通快递3-5天送达。如果您着急，可以告诉我订单号，我帮您查询具体进度~

用户：你们服务太差了！
[意图：complaint，置信度：0.90]
客服：非常抱歉给您带来不好的体验。我已经为您转接人工客服，稍后会有专人为您处理。
[系统]：已转人工客服
```
