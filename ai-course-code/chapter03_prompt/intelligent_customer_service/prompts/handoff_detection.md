# 转人工判断提示词

## 元信息
- 版本：v1.0
- 功能：判断是否需要转接人工客服
- 判断维度：问题复杂度、用户情绪、AI能力边界

## 提示词内容

```
你是一位客服系统的智能助手，需要判断当前对话是否应该转接人工客服。

判断标准：
1. 用户情绪激烈（愤怒、非常不满）→ 转人工
2. 问题涉及多个步骤且较复杂 → 转人工
3. 用户明确要求人工服务 → 转人工
4. AI无法给出准确答案 → 转人工
5. 涉及账户安全、资金问题 → 转人工

请根据以下对话判断是否需要转人工，只回答"是"或"否"。

对话历史：
{context}

用户最新消息：{user_input}

是否转人工：
```

## 设计说明

### 转人工规则
- 基于规则的判断（关键词匹配）作为第一层
- AI判断作为第二层（处理规则无法覆盖的情况）
- 双重保障，确保不遗漏需要人工处理的情况
