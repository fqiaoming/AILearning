# 第5章：LangChain基础

本章代码展示LangChain框架的核心概念和基本用法。

## 文件说明

| 文件 | 说明 | 对应课程 |
|------|------|----------|
| 01_langchain_intro.py | LangChain入门 | 第23课-LangChain入门与核心概念 |
| 02_prompt_template.py | Prompt模板 | 第24课-Prompt-Template深入 |
| 03_output_parser.py | 输出解析器 | 第25课-Output-Parser详解 |
| 04_model_management.py | Model管理 | 第26课-LangChain中的Model管理 |
| 05_lcel_chain.py | LCEL与Chain | 第27课-Chain基础与LCEL深入 |
| 06_sequential_chain.py | 顺序链 | 第30课-SequentialChain |
| project_translator/ | 实战项目：翻译助手 | 第28课-LangChain实战项目1 |

## 使用方法

```bash
# 1. 安装LangChain
pip install langchain langchain-openai

# 2. 确保LM Studio运行或配置OpenAI Key

# 3. 运行示例
python 01_langchain_intro.py
python 05_lcel_chain.py
```
