# 第10章：RAG系统进阶

本章代码展示RAG的高级技巧：HyDE、自查询检索器、上下文压缩等。

## 文件说明

| 文件 | 说明 | 对应课程 |
|------|------|----------|
| 01_hyde.py | HyDE假设文档嵌入 | 第61课 |
| 02_self_query.py | 自查询检索器 | 第62课 |
| 03_context_compression.py | 上下文压缩 | 第63课 |
| 04_parent_document.py | 父文档检索器 | 第64课 |
| 05_multi_strategy_rag.py | 多策略RAG | 第65课 |

## 核心技巧

1. **HyDE**：先生成假设答案，用假设答案去检索
2. **自查询**：自动从问题中提取过滤条件
3. **上下文压缩**：只保留相关的部分
4. **父文档检索**：检索小块，返回大块
